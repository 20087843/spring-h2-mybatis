MyBatis3 Demo Project
===
### 项目说明
本项目是一个基于 h2 Database 和 Mybatis 的 mybatis 使用参考及根据数据库和自定义模板生成代码的参考项目。
其中 generator 代码自动生成模块基于 EasyCode 和 Mybatis Generator 开发，适用于当前主流类型的数据库。
